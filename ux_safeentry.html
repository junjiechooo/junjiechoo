Safe Entry

A more efficient checking out

Problem Statement

We often forgot to check out and by the time we realised it, it is not possible to check out all at once.


Objectives

To implement a button for users to check out selected location or all at once. This aims to improve the usability of the app as users tend to forget to check out whenever they leave a specific location and they had to check out one by one which is time consuming and not user friendly.

Timeline: 1 Week

UX Methods:User Research, User Testing, Design Thinking

Softwares:Adobe Photoshop, Figma

Task: Try checking out all locations that was checked in!
